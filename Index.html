Design sections, share a live room URL, and export the current booth layout as a high-resolution image.

Booth Size

Double click the booth size to edit width and length.

Save Status

Autosave is on. Changes are stored in this browser.

Live Session

Open this app from a hosted URL to collaborate live.

When a room link is active, anyone with that URL can edit the same layout in real time.

Controls

  • Left drag empty booth: create section
  • Two quick left clicks on a section: edit name and size
  • Left drag section: move section
  • Left drag section edge: resize section
  • Right click section: recolor
  • Double right click section: duplicate
  • Arrow keys: nudge selected section by 1 inch
  • Shift + arrow keys: move selected section by 1 foot
  • Mouse wheel: zoom
  • Right drag empty booth: pan
